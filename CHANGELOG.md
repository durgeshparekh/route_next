## 1.0.0

* Initial release.
* URL-based routing with Navigator 2.0.
* Browser refresh preserves current page.
* Deep linking support.
* Auto-syncing drawer, sidebar, and navbar.
* Nested routes and layouts.
* Route guards (middleware).
* Dynamic path parameters and catch-all wildcards.
* Page transition animations.
* document.title updates on web.
* Responsive shell layout (RouteNextScaffold).
